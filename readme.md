# DevCampe API

> Backend API for DevCamper applicatio,which is a bootcamp directory website

## Usage
Rename "config/config.env.env" to "config/config.env" and update the values/setting to your own

## Install dependencies
```
npm install
```

## Run App
```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```
